

\subsection{Distros}
\begin{frame}
\frametitle{Was ist eine Dist.?}
Betrieben von Community/Firmen.
Kümmern sich um: 
\begin{itemize}
 \item Pakete
 \item Änderungen/Patches
 \item Hilfe/Support
\end{itemize}
 
\end{frame}
